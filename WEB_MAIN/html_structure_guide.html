Description / Note

  • <table> 중첩 최대한 지양 (속도저하 발생)
  • 최상위 랩핑 요소 : <div class="opus_design_inquiry"> </div>
  • 각 컬럼(테이블의 세로 단)의 넓이를 그룹핑 할 수 있는 경우 col width="" 속성 / 속성값으로 넓이 조절. 그룹핑 불가 할 경우 <colgroup>,<col> 삭제
  • 조회조건 레이블(타이틀)은 <th>, 데이터는 <td> 로 구분
  • <input type="radio/checkbox"> 태그의 경우 커플 <label> 태그가 있어야 하며, input 의 id 값과 대응하는 커플 label의 for 값이 동일해야 함
  • input_search:단건 조건(돋보기 아이콘 인풋박스) / input_search_btn:돋보기버튼 = 반드시 한줄로 작성(디자인 깨짐)
